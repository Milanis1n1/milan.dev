# Dynamic Portfolio Website

A production-ready, data-driven personal portfolio website built with React, Vite, Tailwind CSS, and Framer Motion.

### Installation

1.  Clone the repository:
    ```bash
    git clone <repository-url>
    cd portfolio
    ```

2.  Install dependencies:
    ```bash
    npm install
    ```

3.  Start the development server:
    ```bash
    npm run dev
    ```

4.  Build for production:
    ```bash
    npm run build
    ```

The application is designed to be **data-driven**. You can update the content without touching the UI code.

Navigate to `src/data` to modify the following files:

-   **`profile.ts`**: Personal details, tagline, and social links.
-   **`skills.ts`**: Technical skills grouped by category.
-   **`experience.ts`**: Professional timeline and achievements.
-   **`projects.ts`**: Featured projects and case studies.
